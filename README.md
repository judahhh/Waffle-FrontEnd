## Waffle

프로젝트 수행을 위한 협업 웹서비스
<br>
<br>
## Description
프로젝트 수행을 위한 협업 웹서비스로, 실시간 채팅, 화상채팅, 그룹, 룸 생성, 게시판 기능, 프로필 기능 등을 제공합니다. <br/>
평소에 자주 사용하던 프로젝트 협업 공간을 다른 컨셉으로 다시 개발해보는 성취를 경험하였습니다. <br/>
